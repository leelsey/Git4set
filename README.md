# Git4set

### Quickely setup git

## Support enviroments
- macOS
- Linux

## Usage
### 1) Initial git setup
If you install new OS, try this:
```bash
sh initial-git.sh
```
### 2) Git configure
If you only setup git basic configure, try this:
```bash
sh git-conf.sh
```
- Branch master to main
- Add user name and email

### 3) Git global ignore
If you want only setup git global ingnore, try this:
```bash
sh git-ignore.sh
```
